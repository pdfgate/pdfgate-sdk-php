# PDFGate's PHP SDK

Official PHP SDK for the PDFGate HTTP API.

PDFGate lets you generate, process, and secure PDFs via a simple API:

- HTML or URL to PDF
- Fillable forms
- Flatten, compress, watermark, protect PDFs
- Extract PDF form data
